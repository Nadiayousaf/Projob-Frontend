# ProJob – Interactive Job Portal (Frontend)

**ProJob** is a modern, interactive, and fully frontend **job portal clone** designed to demonstrate professional UI/UX, animations, and responsiveness. This project is built using **HTML, CSS, and JavaScript** and mimics real-world job websites like LinkedIn or Indeed.

## Features

- **Homepage / Hero Section**
  - Animated hero text and job search bar
  - Call-to-action buttons with hover effects
  - Floating background animations

- **Job Listings Page**
  - Dynamic job cards with slide-in animations
  - Search and filter jobs in real-time (simulated with JS arrays)
  - Interactive pagination and hover effects

- **Company Profiles**
  - Grid layout with hover animations
  - Modal pop-ups for company details
  - Subtle background gradient animation

- **Navigation & Responsiveness**
  - Sticky navigation bar with smooth scroll
  - Responsive design with hamburger menu for mobile
  - Light/Dark mode toggle.

- **Animations**
  - CSS transitions and keyframe animations
  - Scroll-based fade-in and slide-in animations using JS
  - Interactive hover effects on cards and buttons

## Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Flexbox/Grid, transitions, keyframes, responsive design
- **JavaScript** – DOM manipulation, event handling, animations
